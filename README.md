Modern, secure and simple PHP script for mass mailing with user consent.  
Strictly prohibits spam. Requires explicit consent from recipients and confirmation that the sender is 18+.

#### Features
- Sending emails via SMTP (PHPMailer)
- Recipients: manual input or upload (TXT, JSON, PHP)
- Support for attachments
- HTML messages supported
- Multilingual interface (8 languages)
- Mandatory consent checkbox (no spam + 18+)
- Responsive Bootstrap 5 design
- Beautiful styled email template
- Settings page for SMTP configuration
- Donate, Feedback, Terms, About pages
- Error handling and logging
- Content Security Policy (CSP) configured

#### Installation
1. Clone the repository
2. Download [PHPMailer](https://github.com/PHPMailer/PHPMailer) and place the `src/` folder into `PHPMailer/src/`
3. Edit `settings.php` (or use `settings_page.php` in browser)
4. Upload to server (PHP 7.4+ required)
5. Open `index.php`
